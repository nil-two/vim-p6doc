vim-p6doc
=========

Open perl6 document with p6doc.

It's inspired by
[hotchpotch/perldoc-vim](https://github.com/hotchpotch/perldoc-vim).

Usage
-----

#### Modules

	:Perldoc Task::Star

#### Functions

	:Perldoc Mu.say

### Keymap

Type K to open p6doc on the keywords.
